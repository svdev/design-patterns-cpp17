# Design Patterns C++17

Design Patterns Headers-only Library for C++17


## Build

    git clone --recurse-submodules http://github.com/svdev/design-patterns-cpp17
    cd design-patterns-cpp17
    cmake -G "Unix Makefiles" .
    make
    
    
## Test

    make check
